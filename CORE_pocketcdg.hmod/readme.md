=== PocketCDG Core for RetroArch ===

Module adds support for PocketCDG (Karaoke+MP3)

Available executables and arguments to run Core:
- /bin/cdg <rom> <clover_args> 
- /bin/mp3 <rom> <clover_args> 
- /bin/pocketcdg <rom> <clover_args>
